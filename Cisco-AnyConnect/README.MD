## Install Open Connect
```bash
# Packages
apt install lib32z1 lib32ncurses5
apt install openconnect network-manager-openconnect-gnome
apt install openconnect

# Plugins
apt install network-manager-pptp network-manager-vpnc network-manager-openvpn

# for older versions of Ubuntu, this command might be needed.
apt-get install network-manager-openconnect
```
